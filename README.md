# my-first-claude-code

Claude Code を使って作成した初めての Next.js アプリです。

## 技術スタック

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS

## 開発サーバーの起動

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開いて確認できます。

`app/page.tsx` を編集するとページがリアルタイムで更新されます。

## ビルド

```bash
npm run build
npm start
```
